# The valley of tranquility — Vanilla Survival Spawn  
**Minecraft Java Edition (Built in 1.20.1 | Compatible with 1.20+)**

Welcome to **The valley of tranquility**, a vanilla survival map featuring a **custom spawn area** located in a small valley with a lake and a rural cabin with a garden.  
Designed to start a calm survival playthrough and use it as a standard Minecraft world.

---

## Map information
- Type: **Survival / Vanilla / Spawn**
- Base version: **1.20.1**
- Compatibility: **Minecraft Java 1.20+**
- Nether and End: **Vanilla generation (not pre-generated)**

---

## Installation (Minecraft Java)

### Option A — Official Launcher
1) Download the `.zip` file
2) Extract the world folder  
   (it must contain `level.dat`, `region/`, etc.)
3) Copy the folder into:
   - **Windows:** `%appdata%\.minecraft\saves\`
   - **macOS:** `~/Library/Application Support/minecraft/saves/`
   - **Linux:** `~/.minecraft/saves/`
4) Launch Minecraft → *Singleplayer* → select the world

### Option B — Prism Launcher
1) Open Prism Launcher
2) Right-click your instance → **Folder**
3) Open `.minecraft/saves/`
4) Copy the world folder there
5) Launch the instance and enter *Singleplayer*
